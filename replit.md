# C List & Stack Lab

## Overview
This is a university C programming lab assignment focused on implementing List and Stack data structures. Students complete exercises in `exercises.c`.

## Project Structure
- `exercises.c` — Student-editable file with exercise stubs
- `arraylist.h` / `arraylist.c` — List TDA implementation
- `stack.h` — Stack TDA interface
- `test.c` — Test suite (do not modify)
- `test.sh` — Build and run script

## How to Run
Open the Shell and run:
```bash
bash test.sh
```

## Workflow
- **Run Tests** (console) — runs `bash test.sh` to compile and execute tests

## Notes
- No frontend or backend server; this is a pure C console project
- Only `exercises.c` should be modified by students
- GCC 14.3.0 is available via Nix
